# Python Solutions

This directory contains Python implementations of LeetCode problems.

## Structure

Each problem solution will be organized in its own directory, following the pattern:
```
XXXX-problem-name/
  ├── solution.py
  └── README.md
```

## Getting Started

### Requirements
- Python 3.x

### Running Solutions

Navigate to a specific problem directory and run:
```bash
python solution.py
```

## Contributing

When adding a new solution:
1. Create a new directory following the naming convention
2. Add your solution in `solution.py`
3. Include test cases and problem description in `README.md`
